# site-profile

[Link to page](https://fubira.github.io/)

## checkout

```sh
git checkout https://github.com/fubira/site-profile.git
git submodule update --init --recursive
```

## run server

```sh
hugo server
```

## deploy to github

```sh
./deploy.sh [comment]
```

## add new page

```sh
hugo new about.md
```

## add new post

```sh
hugo new posts/post-name.md
# or 
hugo new posts/post-name/index.md
```
