# fubira.github.io

[Link to page](https://fubira.github.io/)

## checkout

```sh
git checkout https://github.com/fubira/fubira.github.io.git
git submodule update --init --recursive
```

## run server

```sh
hugo server
```

## add new page

```sh
hugo new about.md
```

## add new post

```sh
hugo new posts/post-name.md
# or 
hugo new posts/post-name/index.md
```
